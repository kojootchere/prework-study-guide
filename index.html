Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • meta element defines metadata about an HTML document.
  • title element sets the title of the document, displayed in the browser's title bar or tab.
  • h1 to h6 elements represent headings of different levels.
  • p elements represents a paragraph of text.
  • a elements creates a hyperlink to another webpage or resource.
  • img elements display an image on the webpage.
  • ul elements represent an unordered list.
  • ol elements represent an ordered (numbered) list.
  • li elements define a list item in an ordered or unordered list.
  • div elements define a division or section in an HTML document.
  • span elements are an inline container used for grouping and applying styles to a specific part of text.
  • form elements create an interactive form for user input.
  • input elements are used for various form controls, like text fields, checkboxes, and radio buttons.
  • label elements represent a caption for an item in a user interface, often associated with a form control.
  • textarea elements are used to create a multi-line text input control.

CSS

  • Selectors:
    • Element selector selects elements based on their tag names.
    • Class selector selects elements with a specific class name.
    • ID selector selects an element with a specific ID.
    • Attribute selector selects elements based on their attribute values.
    • Pseudo-class selector selects elements in a specific state or condition.
  • Properties:
    • color property sets the text color.
    • background-color property sets the background color.
    • font-family property sets the font for text content.
    • font-size property sets the size of the font.
    • margin property sets the space around an element.
    • padding property sets the space inside an element.
    • display property defines how an element is displayed.
    • position property specifies the positioning of an element.
    • width property sets the width of an element.
    • height property sets the height of an element.
    • border property sets the border of an element.
    • box-shadow property adds a shadow effect to an element.
    • text-align property aligns the text within an element.
    • transition property adds a transition effect to an element.
    • animation property specifies keyframe animations.

Git

  • git init - Initializes a new Git repository.
  • git clone [repository_url] - Creates a local copy of a remote repository.
  • git add [file_name] - Adds a file to the staging area.
  • git commit -m "Commit message" - Commits the changes in the staging area.
  • git status - Displays the current status of the repository.
  • git log - Shows the commit history.
  • git branch - Lists all branches in the repository.
  • git checkout [branch_name] - Switches to a different branch.
  • git merge [branch_name] - Merges changes from one branch into the current branch.
  • git pull - Fetches changes from a remote repository and merges them into the current branch.
  • git push - Pushes local commits to a remote repository.
  • git remote -v - Shows the remote repositories associated with the current repository.
  • git diff - Shows the differences between the working directory and the staging area.
  • git reset [file_name] - Unstages a file from the staging area.
  • git revert [commit_id] - Reverts a specific commit.
  • git stash - Temporarily saves changes that are not ready to be committed.

JavaScript

  • Variables: var, let, and const - used to store and manipulate data
  • Data types: string, number, boolean, null, undefined, object, and symbol - different types of values in JavaScript
  • Operators: arithmetic, assignment, comparison, logical, and ternary - used for mathematical and logical operations
  • Conditional statements: if, else if, else, switch - used to execute different code blocks based on conditions
  • Loops: for, while, do...while, for...in, for...of - used to repeat a block of code multiple times
  • Functions: declaration, expression, arrow functions - reusable blocks of code
  • Arrays: creation, accessing elements, iterating, array methods - used to store and manipulate collections of data
  • Objects: creation, properties, methods, object destructuring - key-value pairs used to represent complex data structures
  • DOM manipulation: selecting elements, modifying content, adding event listeners - interacting with HTML elements on a webpage
  • Asynchronous programming: callbacks, promises, async/await - handling asynchronous operations in JavaScript
  • Error handling: try...catch statements, throwing and catching errors - managing and handling errors in code
  • Modules: exporting and importing modules - organizing code into reusable modules
  • Classes: creating and extending classes - reusable blueprints for creating objects
  • JSON: parsing and stringifying JSON data - working with JSON (JavaScript Object Notation)
  • Browser APIs: localStorage, sessionStorage, Fetch API - accessing browser features and interacting with server data
  • Debugging: using console.log, breakpoints, and browser developer tools - finding and fixing issues in code
