HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • meta element defines metadata about an HTML document.
  • title element sets the title of the document, displayed in the browser's title bar or tab.
  • h1 to h6 elements represent headings of different levels.
  • p elements represents a paragraph of text.
  • a elements creates a hyperlink to another webpage or resource.
  • img elements display an image on the webpage.
  • ul elements represent an unordered list.
  • ol elements represent an ordered (numbered) list.
  • li elements define a list item in an ordered or unordered list.
  • div elements define a division or section in an HTML document.
  • span elements are an inline container used for grouping and applying styles to a specific part of text.
  • form elements create an interactive form for user input.
  • input elements are used for various form controls, like text fields, checkboxes, and radio buttons.
  • label elements represent a caption for an item in a user interface, often associated with a form control.
  • textarea elements are used to create a multi-line text input control.

CSS

  • Class Selector: Use the class selector to target specific elements in CSS.
  • Width Property: Set the width of an element using the "width" property.
  • Margin Property: Adjust the spacing around an element with the "margin" property.
  • Border Property: Customize the border of an element using the "border" property.
  • Box Shadow Property: Apply shadow effects to an element with the "box-shadow" property.
  • Assigning Classes: Use the class attribute in HTML to assign a class selector.
  • Sharing Styles: Assign the same class to multiple elements to share styles.
  • Bullet Points: Create bullet point lists using the "ul" and "li" elements in HTML.
  • Customizing Cards: Style elements as cards by applying a class selector.
  • Background Color Property: Set the background color of an element using "background-color".
  • Text Color Property: Change the color of text with the "color" property.
  • Font Size Property: Adjust the size of text using the "font-size" property.
  • Text Alignment Property: Align text within an element using "text-align".
  • Display Property: Control the display behavior of elements with the "display" property.
  • Padding Property: Set the spacing between the content and the border using "padding".
  • Float Property: Float elements to the left or right using the "float" property.

Git

  • git init - Initializes a new Git repository.
  • git clone [repository_url] - Creates a local copy of a remote repository.
  • git add [file_name] - Adds a file to the staging area.
  • git commit -m "Commit message" - Commits the changes in the staging area.
  • git status - Displays the current status of the repository.
  • git log - Shows the commit history.
  • git branch - Lists all branches in the repository.
  • git checkout [branch_name] - Switches to a different branch.
  • git merge [branch_name] - Merges changes from one branch into the current branch.
  • git pull - Fetches changes from a remote repository and merges them into the current branch.
  • git push - Pushes local commits to a remote repository.
  • git remote -v - Shows the remote repositories associated with the current repository.
  • git diff - Shows the differences between the working directory and the staging area.
  • git reset [file_name] - Unstages a file from the staging area.
  • git revert [commit_id] - Reverts a specific commit.
  • git stash - Temporarily saves changes that are not ready to be committed.

JavaScript

  • Functions in JavaScript are reusable blocks of code that perform specific tasks.
  • To create a function, use the `function` keyword followed by a unique name.
  • Functions need to be called by name to execute their code.
  • Curly braces `{}` contain the code to be executed by the function.
  • Use the `console.log()` method to output messages to the console.
  • The Chrome DevTools console is useful for testing JavaScript code.
  • Debug code by identifying and fixing issues step-by-step.
  • Control flow determines the order in which JavaScript code is executed.
  • Variables store data in named containers for use in code.
  • Conditional statements (`if`, `else if`, `else`) allow for decision-making based on conditions.
  • Loops (e.g., `for`) repeat code execution until a certain condition is met.
  • Arrays store multiple values in a single variable.
  • The `Math` object provides built-in mathematical functions in JavaScript.
  • Use `Math.floor()` to round a number down to the nearest integer.
  • Use `Math.random()` to generate a random number between 0 (inclusive) and 1 (exclusive).
